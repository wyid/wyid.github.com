  2. 修改ip地址：/etc/sysconfig/network-script/网卡，修改onboot=yes

  3. BOOTPROTO=dhcp -->BOOTPROTO=static IP获取方式改为静态获取 ZONE=public firewalld zone=piblic（公共）：在公共区域内使用，不能相信网络内其他计算机不会对你造成危害，只能接受经过选取的连接。 PADDR=10.82.17.71 IP地址 ATEWAY=10.82.17.1 网关 ，与虚拟机虚拟网卡VMnet8中设置的网关保持一致 ETMASK=255.255.255.0 子网掩码 DNS1=10.82.1.4 DNS2=10.82

  4. 安装wget

  5. 配置源

  6. 安装vim

  7. 重启网络：service network restart
