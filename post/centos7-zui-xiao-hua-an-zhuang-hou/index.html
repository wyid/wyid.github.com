CentOS7最小化安装后

2019-07-05 10:07 #Linux

centos7选择最小化安装后，配置过程：

查看ip地址：ip addr/ifconfig

修改ip地址：/etc/sysconfig/network-script/网卡，修改onboot=yes BOOTPROTO=dhcp -->BOOTPROTO=static IP获取方式改为静态获取 ZONE=public firewalld zone=piblic（公共）：在公共区域内使用，不能相信网络内其他计算机不会对你造成危害，只能接受经过选取的连接。

PADDR=10.82.17.71 IP地址 ATEWAY=10.82.17.1 网关 ，与虚拟机虚拟网卡VMnet8中设置的网关保持一致 ETMASK=255.255.255.0 子网掩码 DNS1=10.82.1.4 DNS2=10.82

安装wget 配置源 安装vim 重启网络：service network restart 修改主机名：hostname/hostnamectl hostnamectl set-hostname centos vim /etc/hostname 自动补全：yum install bash-completion 查询jdk安装：rpm -qa|grep java/jdk/gcj 卸载jdk：rpm -e --nodeps
配置环境变量：/etc/profile export JAVA_HOME=/opt/jdk1.8.0_144 export CLASSPATH=.:JAVAHOME/jre/lib/rt.jar:JAVA_HOME/jre/lib/rt.jar:JAVAH​OME/jre/lib/rt.jar:JAVA_HOME/lib/dt.jar:JAVAHOME/lib/tools.jarexportPATH=JAVA_HOME/lib/tools.jar export PATH=JAVAH​OME/lib/tools.jarexportPATH=PATH:$JAVA_HOME/bin source /etc/profile

挂载/解除：mount -t auto /dev/cdrom /mnt/cdrom umount /dev/cdrom
